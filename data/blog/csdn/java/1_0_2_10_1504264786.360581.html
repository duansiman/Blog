java区别于c语言，很重要的一点就在于java是一门面向对象的编程语言。这里就先说说对对象和类的一些理解。

1.简单的解释

在自然界中，对于任何一样东西我们都可以理解为对象，万物皆对象。而类是指对一些对象的总称，如我们看到两只麻雀都会想到它们是一类的东西，都是鸟类。但是对于它们个体来说，它们是不同的对象

2.简单实例

public class Dog{
    static int snum=0;  //表示狗的总数
    int num=0;          //数量，用来和static比较
    String name;
    public Dog(String name){
        this.name=name;
        num++;
        snum++;
    }
}

这里我们定义了一个名字为Dog的类，有三个属性，名称，数量和总数，然后又定义了一个有参的构造方法。

public class MyTest{
    public static void main(String[] args){
        Dog dog1=new Dog("HaPi");
        Dog dog2=new Dog("LiLi");
        System.out.println(dog1.name+":"+dog1.num);
        System.out.println(dog2.name+":"+dog2.num);
        System.out.println(Dog.snum);
    }
}

这里读者们在测试前可以先自己想想输出结果是什么，再去测试，看看输出结果是否和自己所想的相同。

这里有点需要说明，如果我们不定义自己的构造方法，它自己会默认给Dog一个无参的构造方法

public Dog{super();}

上面输出结果应该如下

HaPi:1
LiLi:1
2

我们在测试代码中定义了”HaPi”和”LiLi”两只狗，他们就是两个对象，而他们都是Dog类的对象。所以他们有自己的名字，分别是”HaPi”和”LiLi”，然后他们自己的数量当然应该都是1了(我这么说三头地狱犬怎么办……)。

3.static关键字的一个简单理解

在上面例子中，Dog类中的num属性是没有加static的，这就说明num是属于对象的而不是属于一个类的。加上static则表示这个属性是属于Dog类的。（这么说可能不太理解，请看下面例子）

Dog.snum;  //编译通过
Dog.name;   //报错
Dog dog=new Dog("123");
dog.snum;  //报错
dog.name;   //编译通过

从这几个比较，我们可以很清楚的理解static和非static的区别。
Dog.snum;相当于表示狗的总数
Dog.name;狗这个物种的具体名字，这里Dog是一个类没有具体名字，就报错了。我们可以通过下面代码获取类全名称

System.out.println(Dog.class.getName());

dog.snum; 由于狗的总数是属于狗这个物种的，所以我们无法通过一条狗来知道世界上狗的总数。
dog.name;对于一条狗，在起名字后，我们就可以直接叫它名字
