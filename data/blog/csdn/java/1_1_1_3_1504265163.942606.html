用 jsoup做爬虫采集站点

本章任务

* 获取百度新闻首页
* 获取实时热点的第一页的链接

获取百度新闻首页

* 从一个URL加载一个Document

Document doc = Jsoup.connect("https://www.baidu.com/").get();
String content = doc.html();

content 就是我们获得的百度新闻首页的文档数据

获取实时热点的第一页的链接

实时热点的新闻链接：首先是在ul标签里，它的class为s-news-rank-content；其次链接标签是a，它的class为title-content

<ul class="s-news-rank-content">

<li class="news-meta-item news-artist" data-index="0">
<span class="news-title"><span class="title-link"><a class="title-content" href="https://www.baidu.com/s?cl=3&tn=baidutop10&fr=top1000&wd=%E4%B8%96%E7%95%8C%E8%A5%BF%E5%95%86%E5%A4%A7%E4%BC%9A%E4%B8%BE%E5%8A%9E&rsv_idx=2" target="_blank">世界西商大会举办</a><span class="new-identity"></span></span></span>
</li>

...
...
</ul>
Elements alinks = doc.select("ul.s-news-rank-content a[href].title-content");
for (Element link: alinks ) {
       System.out.println(  link.text() + "||"  + link.attr("abs:href") );
}

这样我们就获取了链接的名字和地址
