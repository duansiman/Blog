概述

今天在打开eclipse，编辑Android代码时，发现没有自动提示。

在经过百度搜索并验证后，总结一下，方法分为两步。

1、设置“Auto activation triggers for Java”

2、设置“Java Proposals”

一、设置"Anto activation triggers for java"

打开：Eclipse -> Window -> Perferences -> Java -> Editor -> Content Assist ->Auto activation triggers for Java：

将原来的"."改成"abcdefghijklmnopqrstuvwxyz."。


二、设置“Java Proposals”

打开：Eclipse ->window->Preferences->Java->Editor->Content Assist->Advanced->顶部的选项卡Select the proposal kinds contained in the 'default' content assist list: 中把”Java Proposals“ 选项打上勾。


到此，可以解决Eclipse Android编辑时，不能自动提示的问题。
