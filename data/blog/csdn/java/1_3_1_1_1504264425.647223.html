基于MySQL数据库的sql语句

数据库定义语句：

1.创建一个表格

create table book (

id int primary key,

name varchar(30),

age int

);


2.删除一个表格

drop table book

3.修改一个表格

alter table book

4.显示数据库中的表格

show tables

desc tables


数据库操作语句：(增删改查)

1.插入一行记录（数据）

insert into book () values ();

2.删除一行记录

delete from book where id='';

3.更新一行记录

update book set name='王五' where id=4;

4.查询指定表格记录

select * from book where id=2;
