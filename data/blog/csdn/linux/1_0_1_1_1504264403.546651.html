    最近项目中使用solr提供全文检索服务，研究了研究，从安装开始使用Solr。

因为Solr是一个使用java开发的web项目，启动solr服务需要提供web容器、JDK两个java web环境，下面是JDK、Tomcat、Solr在Linux环境下的安装。

一、Linux 安装JDK

使用jdk-7u55-linux-i586.tar.gz 在Liunx系统中安装JDK1.7 :

1、上传解压

将JDK的tar压缩包上传到服务器，并解压命令：

  tar -zxvf /usr/local/jdk-7u55-linux-i586.tar.gz

2、配置JDK环境变量

[root@localhost local]# sudo vi /etc/profile

在配置文件末尾添加：

	#set java environment
	JAVA_HOME=/usr/local/jdk1.7.0
	CLASSPATH=.:$JAVA_HOME/lib.tools.jar
	PATH=$JAVA_HOME/bin:$PATH
	export JAVA_HOME CLASSPATH PATH
3、使用source /etc/profile 命令使刚修改的配置生效

4、使用java -version查看JDK安装，显示版本号1.7.0_55-JDK安装成功

      

二、Linux 安装Tomcat

1、同样，首先上传tomcat压缩包并解压

2、为方便管理，创建指定目录，并将解压后的tomcat 拷贝到该目录 

  1. Tomcat即安装成功

    三、Linux 安装Solr

    1、上传解压，将解压后的solr.war文件拷贝到tomcat中

    [root@localhost dist]# cp solr-4.10.3.war /usr/local/solr/tomcat/webapps/solr.war

    2、启动tomcat，自动解压solr的war包

         

       3、将gz包解压后的solr/example实例中提供的solrhome 文件拷贝到usr/local/solr/solrhome中

        

        4、把solr所需的日志依赖jar 拷贝到tomcat/webapp/solr/lib下，并修改web.xml配置文件，指定solrhome路径

       

        5、再次启动tomcat，访问192.168.21.48：8080/solr ,显示solr后台管理页面，solr安装发布成功。
