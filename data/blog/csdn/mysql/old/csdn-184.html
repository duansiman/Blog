   用mybatis在mysql中保存字段精确到秒需要两个步骤。

\n

1、现在mysql中将时间字段改为datetime 例如：alter table tablename add pay_date \ndatetime comment '精确到秒的时间'
\n

\n

2、在sql语句中直接用now().  例如：        update tj_record set   is_customerHandover ='1' and customer_handover_date = now() where id = #{record_id}
