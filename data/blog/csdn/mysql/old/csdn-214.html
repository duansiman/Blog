RPM方式安装MySQL5.6

\n

1.检查MySQL及相关RPM包
\n

\n

检查MySQL及相关RPM包，是否安装，如果有安装，则移除（rpm –e 名称）

\n

\n
[root@localhost ~]# rpm -qa | grep -i mysql\nmysql-libs-5.1.66-2.el6_3.x86_64\n[root@localhost ~]# yum -y remove mysql-libs*
\n

\n

2.下载rpm包

\n

 下载Linux对应的RPM包，如：CentOS6.4_64对应的RPM包，如下：
\n

\n
[root@localhost rpm]# ll\ntotal 74364\n-rw-r--r--. 1 root root 18442536 Dec 11 20:19 MySQL-client-5.6.15-1.el6.x86_64.rpm\n-rw-r--r--. 1 root root 3340660 Dec 11 20:06 MySQL-devel-5.6.15-1.el6.x86_64.rpm\n-rw-r--r--. 1 root root 54360600 Dec 11 20:03 MySQL-server-5.6.15-1.el6.x86_64.rpm
\n

\n

3. 安装MySQL

\n

\n
[root@localhost rpm]# rpm -ivh MySQL-server-5.6.15-1.el6.x86_64.rpm\n[root@localhost rpm]# rpm -ivh MySQL-devel-5.6.15-1.el6.x86_64.rpm\n[root@localhost rpm]# rpm -ivh MySQL-client-5.6.15-1.el6.x86_64.rpm\n#修改配置文件位置\n[root@localhost rpm]# cp /usr/share/mysql/my-default.cnf /etc/my.cnf
\n

\n

4. 初始化MySQL及设置密码

\n

\n
[root@localhost rpm]# /usr/bin/mysql_install_db\n[root@localhost rpm]# service mysql start\n[root@localhost rpm]# cat /root/.mysql_secret #查看root账号密码\n# The random password set for the root user at Wed Dec 11 23:32:50 2013 (local time): qKTaFZnl\n[root@localhost ~]# mysql -uroot –pqKTaFZnl\nmysql> SET PASSWORD = PASSWORD('123456'); #设置密码为123456\nmysql> exit\n[root@localhost ~]# mysql -uroot -p123456
\n

\n

5. 允许远程登陆

\n

\n
mysql> use mysql;\n\nmysql> select host,user,password from user;\n\n+-----------------------+------+-------------------------------------------+\n\n| host | user | password |\n\n+-----------------------+------+-------------------------------------------+\n\n| localhost | root | *6BB4837EB74329105EE4568DDA7DC67ED2CA2AD9 |\n\n| localhost.localdomain | root | *1237E2CE819C427B0D8174456DD83C47480D37E8 |\n\n| 127.0.0.1 | root | *1237E2CE819C427B0D8174456DD83C47480D37E8 |\n\n| ::1 | root | *1237E2CE819C427B0D8174456DD83C47480D37E8 |\n\n+-----------------------+------+-------------------------------------------+\n\n \n\nmysql> update user set password=password('123456') where user='root';\n\nmysql> update user set host='%' where user='root' and host='localhost';\n\nmysql> flush privileges;\n\nmysql> exit
\n

\n

6. 设置开机自启动

\n

\n
[root@localhost ~]# chkconfig mysql on\n\n[root@localhost ~]# chkconfig --list | grep mysql\n\nmysql 0:off 1:off 2:on 3:on 4:on 5:on 6:off
\n

\n

7. MySQL的默认安装位置

\n

\n
/var/lib/mysql/ #数据库目录\n\n/usr/share/mysql #配置文件目录\n\n/usr/bin #相关命令目录\n\n/etc/init.d/mysql #启动脚本
\n

\n

8.修改字符集和数据存储路径

\n

配置/etc/my.cnf文件,修改数据存放路径、mysql.sock路径以及默认编码utf-8.
\n

\n

\n
[client]\npassword = 123456\nport = 3306\ndefault-character-set=utf8\n[mysqld]\nport = 3306\ncharacter_set_server=utf8\ncharacter_set_client=utf8\ncollation-server=utf8_general_ci\n#(注意linux下mysql安装完后是默认：表名区分大小写，列名不区分大小写； 0：区分大小写，1：不区分大小写)\nlower_case_table_names=1\n#(设置最大连接数，默认为 151，MySQL服务器允许的最大连接数16384; )\nmax_connections=1000\n[mysql]\ndefault-character-set = utf8
\n

\n

9.查看字符集

\n

\n
show variables like '%collation%';\n\nshow variables like '%char%';
\n
