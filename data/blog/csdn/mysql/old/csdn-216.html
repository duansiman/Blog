记得那时因为没有网络，把root密码改错了写成了：
\nupdate user set password="122" where user="root";
\n之后mysql就开始不工作了。
\n#mysql -u root -p
\n#123
\n就会出现：ERROR 1045 (28000): Access denied for user ''@'localhost' (using password: NO)
\n只能这样：
\n#mysql -u root
\nmysql> use mysql
\n又出现了ERROR 1045 (28000): Access denied for user ''@'localhost' (using password: NO)
\n找了好久资料：
\n解决方法：

\n

1.关闭mysql
\n   # service mysqld stop
\n2.屏蔽权限
\n   # mysqld_safe --skip-grant-table
\n   屏幕出现： Starting demo from .....
\n3.新开起一个终端输入
\n   # mysql -u root mysql
\n   mysql> UPDATE user SET Password=PASSWORD('newpassword') where USER='root';
\n   mysql> FLUSH PRIVILEGES;//记得要这句话，否则如果关闭先前的终端，又会出现原来的错误
\n   mysql> \\q
\n这样就好了。
