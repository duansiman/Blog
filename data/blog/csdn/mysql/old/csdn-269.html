       UNIQUE KEY

\n

       唯一约束要求：唯一约束可以保证记录的唯一性；唯一约束的字段可以为空值(NULL)；每张数据表可以存在多个

\n

\n

       例子：

\n

       CREATE TABLE tb5(

\n

          id SMALLINT UNSIGNED AUTO_INCREMENT PRIMARY KEY,

\n

          username VARCHAR(30) NOT NULL UNIQUE KEY,

\n

          age TINYINT UNSIGNED

\n

       );

\n
