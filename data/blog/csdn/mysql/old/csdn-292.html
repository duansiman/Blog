《Ask Oracle官方原创》

\n

Oracle官方文档数量庞大，而且往往没有侧重点，让oracle新手看起来很费力。但是，仍有很多Oracle使用者认为任何oracle学习资料都比不上Oracle官方文档的权威和扼要，且兼具基础与全面。这种差异可能与个人的阅读方法有很大关系，本文将为您提供一些可供参考的Oracle官方文档阅读方法（这里以10g \n R2为主）。可能有些同学还不知道Oracle官方文档的下载链接这里做个小提示：

\n

Oracle 10 R2相关：Oracle目前已经不支持Oracle 10g版本的下载，这里只提供在线阅读链接（纯属友情提供）《Oracle \n Database，10g Release 2 (10.2)在线阅读》

\n

Oracle 11g R2相关：《Oracle \n Database, 11g Release 2 (11.2)》下载&《Oracle Database, 11g Release 2 (11.2)在线阅读》

\n

Oracle 12c R1相关：《Oracle \n Database, 12c Release 1 (12.1)》下载 &《Oracle Database, 12c Release \n 1 (12.1)在线阅读》

\n

Reference
\n

\n

Reference意思是参考，向产品使用说明书一样包罗万象，建议无需精读，碰到问题时速查即可

\n

Concepts【必读】

\n

Concepts是Oracle基础入门必读文档入门级，系统级地读物。

\n

Oracle Database Concepts

\n

Oracle \n Database Concepts可以使你了解Oracle、RDBMS的相关概念和体系结构,还可以了解到其它的文档中提及的名词和概念。如果你从未接触过Oracle，这个文档是很好的入门读物。本书网友们极力推荐要精读。

\n

Oracle Database New Features Guide

\n

每次新版本的Oracle RELEASE以后，看一下《Oracle \n Database New Features Guide》，以了解一下最新的FEATURE，此文档可以粗略地阅读，不需精读。

\n

Oracle Database Administrator’s Guide【必看+精读】

\n

Oracle数据库管理。作为DBA，《Oracle \n Database Administrator’s Guide》是必须要看的一本书

\n

Oracle Database Reference

\n

Oracle \n Database Reference讲述Oracle的所有初始化参数，数据字典，动态性能视图，Oracle自带的用于数据库管理的SQL脚本，Oracle等待事件，Oracle统计信息等知识。无需精读。

\n


\n

\n

基于平台的管理

\n

Oracle \n Database Administrator’s Reference for Linux and UNIX-Based Operating Systems讲述在类UNIX平台如何管理Oracle数据库，如设置环境变量，内核参数等,这篇文章是在BOOK这个目录下的。

\n

Oracle备份和恢复

\n

Oracle \n Database Backup and Recovery User’s Guide【精读】。作为DBA，精通数据库的备份和恢复是没得说的，这篇文章是在BOOK这个目录下的。

\n

Oracle \n Database Backup and Recovery Reference，这个文档提供ORALCE备份和恢复管理器（RMAN）的命令的速查。无需精读，这篇文章是在BOOK这个目录下的。

\n

网络管理与配置

\n

Oracle \n Database Net Services Administrator’s Guide，Oracle企业管理器安装与配置

\n

Enterprise \n Manager Concepts，这个文档可以使你了解EM，入门级读物

\n

Enterprise \n Manager Grid Control Installation and Basic Configuration，这个文档是告诉你怎么安装Oracle的Grid Control的，包括Management Agent的安装

\n

Enterprise \n Manager Advanced Configuration，这个文档讲述Oracle企业管理器的高级配置

\n

Oracle实用工具

\n

Oracle \n Database Utilities，这个文档介绍Oracle常用且实用的工具：DataPump,SQL*Loader,DBVERIFY,DBNEWID,LogMiner,MetaData API,Export&Import,ASMCMD等的使用

\n

Oracle数据库开发相关

\n

SQL \n Developer User's Guide，PL/SQL User's Guide \n and Reference，这个文档主要讲述如何利用SQL和PL/SQL来进行数据库开发。

\n

Oracle \n Database SQL Reference，全面介绍SQL语言。此文档无需精读，碰到不懂的地方查看此文档即可

\n

Oracle \n Database PL/SQL Packages and Types Reference，这个文档全面介绍Oracle数据库中内置的PL/SQL包和类型，如比较常用的以CTX_（Oracle TEXT相关的包）DBMS_（Oracle数据库管理相关的包），UTL_（Oracle实用工具相关的包）

\n

SQL*Plus \n User’s Guide and Reference，这个文档介绍Oracle常用的客户端工具SQL*Plus的使用

\n

Oracle全球化支持

\n

Oracle \n Database Globalization Support Guide，这个文档讲述全球化支持（ITN），字符集，排序规则等。此外还介绍了Character Set Scanner和Oracle Locale Builder等实用工具的使用

\n

数据库性能优化

\n

Oracle \n Database Performance Tuning Guide，全面介绍如何优化Oracle数据库。

\n
