      聚合函数

\n

      常见的聚合函数：

\n


\n

\n

      聚合函数有一个共同点就是只有一个返回值。

\n

      (1)使用AVG()函数求取平均值

\n

      USE t2;

\n

      SHOW TABLES;

\n

      SELECT ROUND(AVG(goods_price),2) AS avg_price FROM tdb_goods;

\n


\n

\n

      (2)使用计数函数、最大值函数以及最小值函数

\n

      SELECT COUNT(goods_id) AS counts FROM tdb_goods;

\n      SELECT MAX(goods_price) AS max_price FROM tdb_goods;
\n \n

      SELECT MIN(goods_price) AS min_price FROM tdb_goods;

\n


\n

\n

      (3)使用SUM()函数求和

\n

      SELECT SUM(goods_price) AS sum_price FROM tdb_goods;

\n

       使用加密函数的例子

\n

       SELECT MD5('admin');

\n

\n


\n

\n

       使用加密函数的建议：

\n

       如果是为了以后的Web页面做准备，那我们尽量使用MD5()；PASSWORD()只有一个用途，就是我们在修客户端

\n

自己密码的时候可以使用。

\n

       修改客户端密码的格式：

\n       SET PASSWORD=PASSWORD('123123');
