2、配置两台Mysql主主同步

\n

关于MySQL的安装大家也可以参考《 MySQL之——CentOS6.5 编译安装MySQL5.6.16 》，主主同步在主从同步的基础上将从服务器配置为之前主服务器的Master，相当于在原来主从同步的基础上，将原来的Slave设置为了原来Master的Master，大家也可以参考《MySQL之——MS主从复制(读写分离)实现》 \n ，设置A为B的Master，B为A的Slave后，然后再将B设置为A的Master,A设置为B的Slave。
\n

\n
[root@masterr ~]# yum install mysql-server mysql -y\n[root@masterr ~]# service mysqld start\n[root@masterr ~]# mysqladmin -u root proot\n[root@masterr ~]# vi /etc/my.cnf #开启二进制日志，设置id\n[mysqld]\nserver-id = 1 #backup这台设置2\nlog-bin = mysql-bin\nbinlog-ignore-db = mysql,information_schema #忽略写入binlog日志的库\nauto-increment-increment = 2 #字段变化增量值\nauto-increment-offset = 1 #初始字段ID为1\nslave-skip-errors = all #忽略所有复制产生的错误 \n[root@masterr ~]# service mysqld restart
