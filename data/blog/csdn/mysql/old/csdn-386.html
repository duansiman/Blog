本文主要介绍centos下源码安装MySQL 5.6的方法，centos的版本为5.8.

\n

1）首先，你需要到MySQL官网下载源码tar包，点击MySQL Community Server，选择Source Code，源码包不大，只有34M左右。

\n

注：以下操作没有特殊说明，都是以root账户执行。

\n

2）先安装cmake（mysql5.5以后源码安装都得通过cmake编译）

\n

\n
# yum install cmake
\n

\n

并确保以下两个包已安装最新版：

\n

  ncurses
\n  ncurses-devel
\n

\n3）添加MySQL用户组和用户 \n

\n
# groupadd mysql\n# useradd -g mysql mysql
4）创建MySQL软件安装目录和数据存放目录 \n

\n

\n
mkdir -p /opt/mysql #MySQL安装目录\nchown -R mysql:mysql /opt/mysql\nmkdir -p /data/mysql #MySQL数据存放目录\nmkdir -p /data/mysql/data #存放数据\nmkdir -p /data/mysql/log #存放日志\nmkdir -p /data/mysql/tmp #存放临时文件\nchown -R mysql:mysql /data/mysql
5）cmake编译安装MySQL \n

\n

\n
tar zxvf mysql-5.6.13.tar.gz\ncd mysql-5.6.13\ncmake . -DCMAKE_INSTALL_PREFIX=/opt/mysql -DMYSQL_DATADIR=/data/mysql -DSYSCONFDIR=/opt/mysql -DDEFAULT_CHARSET=utf8 -DDEFAULT_COLLATION=utf8_general_ci -DEXTRA_CHARSETS=all -DENABLED_LOCAL_INFILE=1\nmake\nmake install
\n

\n

参数说明：
\n

\n

-DCMAKE_INSTALL_PREFIX=/opt/mysql        //指定安装目录
\n-DINSTALL_DATADIR=/data/mysql            //指定数据存放目录
\n-DSYSCONFDIR=/opt/mysql                  //指定配置文件目录（本例的配置文件为/opt/mysql/my.cnf）
\n-DDEFAULT_CHARSET=utf8                   //指定字符集
\n-DDEFAULT_COLLATION=utf8_general_ci      //指定校验字符
\n-DEXTRA_CHARSETS=all                     //安装所有扩展字符集
\n-DENABLED_LOCAL_INFILE=1                 //允许从本地导入数据
\n

\n

上述步骤执行成功后，MySQL软件已成功安装至/opt/mysql目录下。

\n

8）创建数据库

\n

\n
/opt/mysql/scripts/mysql_install_db --user=mysql
\n

\n

上述建库语句将根据my.cnf里设置的数据文件目录和日志文件目录，生成相应的数据文件和日志文件，并创建系统数据库（如mysql,test,information_schema,performance_schema)

\n

9）启动MySQL

\n

\n
/opt/mysql/support-files/mysql.server start
\n

\n

启动成功后，就可以以root用户登录（默认的root用户没有密码）

\n

\n
/opt/mysql/bin/mysql -u root
\n
