1）创建数据库：
\n使用if exists 来判断是否已经存在，不存在，才会创建，已经存在，则不创建，并且不会报错，如果没有用if exists ，再次创建相同名字的数据库，就会报错！
\n Case: create database if not exists new08;
\n

\n\n

2)显示数据库：
\nshow命令,显示已经建好的数据库
\nCase: show databases;
\n可以看到所有的建好的数据库；
\n如果我想模糊查询以“new..”开头的数据库，可以用like 关键字：
\nCase: show databases like ‘new%’

\n\n

注意事项：
\n01–%是通配符，代表一个或者多个字符；
\n02–_下划线是通配符，代表一个字符；
\n03–用show命令的时候，切记，后面的databases是复数形式，上次网上看到一个帖子，一哥们少写个s纠结好久，咳咳····

\n\n

3)选用数据库
\n语法：use +database
\nCase: use new01;
\n会出现一行‘Database changed’的友情提示；
\n

\n\n

4)删除数据库
\n使用drop命令
\n语法：drop database +数据库名字；
\nCase: drop database new01;

\n\n

关注一下if exists 命令；
\nCase：drop database if exists new04;
\n这样加上if exists 的好处是为了避免new04数据库不存在，然后报错误信息，
