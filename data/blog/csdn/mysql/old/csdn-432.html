使用MySQL自带的命令行备份、还原

\n\n

假定mysql安装在D:/software/MySQL Server 5.1/
\n那么其bin目录下有一个mysqldump.exe文件
\n账号是root
\n密码是admin
\n备份的数据库名是 test
\n备份后的文件是 d:\\test.sql

\n\n

备份

\n\n
"D:/software/MySQL Server 5.1/bin/mysqldump.exe" -uroot -padmin -hlocalhost -P3306 test -r d:\\test.sql
\n\n\n\n

还原

\n\n
"D:/software/MySQL Server 5.1/bin/mysql.exe" -u root -padmin test < d:/test.sql
\n\n\n\n

innodb开启*【特别注意MyISAM不支持事务】*

\n\n

检查是否开启

\n\n
SHOW variables like 'have_%';
\n\n

\n\n\n\n

停止MySQL

\n\n
net stop mysql
\n\n

定位到D:\\tools\\MYSQL\\mysql-5.1.57-win32\\data 目录下，找到这3个文件
\nib_logfile0
\nib_logfile1
\nibdata1
\n备份后，删除掉

\n\n

注： D:\\tools\\MYSQL\\mysql-5.1.57-win32 是我的安装目录，请自行调整

\n\n\n\n

修改配置文件

\n\n

打开文件：D:\\tools\\MYSQL\\mysql-5.1.57-win32\\my.ini

\n\n
添加一行 \ndefault-storage-engine=InnoDB\n\n如果已经存在\ndefault-storage-engine=MyISAM\n\n像下面这样，注释掉即可\n#default-storage-engine=MyISAM
\n\n\n\n

启动MySQL

\n\n
net start mysql
