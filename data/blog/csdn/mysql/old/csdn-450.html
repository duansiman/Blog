1、创建数据库：create Database databaseName;例如需要创建mysqlTest数据库，create Database mysqlTest;

由此还可以看出，这里的名称是不分大小写的，据我所知可以通过设置设置为全大些或者全小写。

2、查看系统中已经存在的数据库：show databases;


3、选择数据库：use databaseName;(本文中所有databaseName均是一个代称，表示数据库名称，tableName也是代称，表示表名)；


4、删除数据库：drop database databaseName;


5、查看数据库引擎：show engines  \\G


支持的引擎比较多，因此这里屏幕截图就不完整。

6、查看数据库默认存储引擎：show variables like 'storage_engine%';


7、查看系统帮助：help contents;


8、创建表：create table tableName(
   属性名 数据类型，
   属性名 数据类型，
   ......
)


9、查看表结构：describe tableName；


10、删除表：drop table tableName；
