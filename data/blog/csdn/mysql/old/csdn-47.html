\nCEIL和FLOOR
\n
\nCEIL为向上取整，只要有小数位，不管是几，直接入一位。例如：
\n
mysql> SELECT CEIL(3.01);\n+------------+\n| CEIL(3.01) |\n+------------+\n| 4 |\n+------------+\n1 row in set (0.02 sec)\n
\n

\n
\nFLOOR与此 \n

相反
\n

\n
\n
mysql> SELECT FLOOR(3.99);\n+-------------+\n| FLOOR(3.99) |\n+-------------+\n| 3 |\n+-------------+\n1 row in set (0.00 sec）\n
\n

\n
\nDIV和MOD
\n
\nDIV为取整，MOD为取余
\n

\n

mysql> SELECT 3/4;\n+--------+\n| 3/4 |\n+--------+\n| 0.7500 |\n+--------+\n1 row in set (0.00 sec)\n\nmysql> SELECT 3 DIV 4;\n+---------+\n| 3 DIV 4 |\n+---------+\n| 0 |\n+---------+\n1 row in set (0.00 sec)\n\nmysql> SELECT 3 MOD 4;\n+---------+\n| 3 MOD 4 |\n+---------+\n| 3 |\n+---------+\n1 row in set (0.00 sec)

\n \n

\n
\n注：MOD可以换成％
\n
\n

\n
\nPOWER幂运算
\n

\n

例如3的2次方

\n

\n

mysql> SELECT POWER(3,2);\n+------------+\n| POWER(3,2) |\n+------------+\n| 9 |\n+------------+\n1 row in set (0.14 sec)\n

\nROUND四舍五入
\n

\n

\n
\n例如3.652保留小数点后两位。
\n

\n

mysql> SELECT ROUND(3.652,2);\n+----------------+\n| ROUND(3.652,2) |\n+----------------+\n| 3.65 |\n+----------------+\n1 row in set (0.00 sec)\n
\n

\n

\n
\n保留小数点后一位：
\n

\n

mysql> SELECT ROUND(3.652,1);\n+----------------+\n| ROUND(3.652,1) |\n+----------------+\n| 3.7 |\n+----------------+\n1 row in set (0.00 sec)\n
