\n1，MySQL：(structured query  language)用于访问和处理数据库的标准语言     
\n
\n2，什么是 SQL？
\n
\n     SQL 指结构化查询语言
\n
\n     SQL 使我们有能力访问数据库（数据库是按照数据结构来组织，存取和管理数据的仓库）
\n
\n     SQL 是一种 ANSI 的标准计算机语言
\n
\n3，SQL 能做什么？
\n
\n     SQL 面向数据库执行查询
\n
\n     SQL 可从数据库取回数据
\n
\n     SQL 可在数据库中插入新的记录
\n
\n     SQL 可更新数据库中的数据
\n
\n     SQL 可从数据库删除记录
\n
\n     SQL 可创建新数据库
\n
\n     SQL 可在数据库中创建新表
\n
\n     SQL 可在数据库中创建存储过程
\n
\n     SQL 可在数据库中创建视图
\n
\n     SQL 可以设置表、存储过程和视图的权限
\n

\n4，在终端上常用的指令(基于桌面的文件管理)
\n
\n     touch + 文件名.文件类型    //创建一个文件
\n
\n     open + 文件名                 //打开 对应的文件或者是文件夹 
\n
\n     ls                                   //列出所有的文件
\n
\n     clear                              //清屏记录
\n
\n     mkdir + 目录的名字           //创建目录(文件)
\n
\n     cp + 参数 + 源文件 + 目标文件  + 拷贝  //移动文件
\n
\n     rm+ 参数 +文件              //删除文件
\n
\n     cd ../                            //返回上一级目录
\n
\n     ping                              //给当前的网络主机发送一个 回应请求
\n
\n     mail                              //阅读和发送电子邮件
\n
\n     uname                          //显示操作系统的有关信息
\n
\n     who                              //列出当前登录所有的用户
\n
\n     whoami                         //显示当前正在操作的用户名
\n
\n     df                                 //显示文件的系统的总空间和可用空间     
\n
\n     w                                 //显示当前的这个控制台
\n
\n     history                          //列出最近执行过的 几条命令及编号
