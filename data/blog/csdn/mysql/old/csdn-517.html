1、加载JDBC驱动程序：

\n\n
在连接数据库之前，首先要加载想要连接的数据库的驱动到JVM（Java虚拟机），这通过java.lang.Class类的静态方法forName(String className)实现。 \n
\n\n\n\n
 //加载数据库驱动总共有三种方式：\n try{ \n //① \n Class.forName("com.mysql.jdbc.Driver") ; \n //②\n com.mysql.jdbc.Driver driver = new com.mysql.jdbc.Driver();\n //③\n new com.mysql.jdbc.driver();\n\n }catch(ClassNotFoundException e){ \n System.out.println("找不到驱动程序类 ，加载驱动失败！"); \n e.printStackTrace() ; \n }
\n\n

成功加载后，会将Driver类的实例注册到DriverManager类中。

\n\n\n\n

2、提供JDBC连接的URL

\n\n
\n
    \n
  • 连接URL定义了连接数据库时的协议、子协议、数据源标识。
    • \n
  • MYSQL的JDBC URL 编写方式：jdbc:mysql://主机名称：端口号/数据库名称？user = 用户名 & password =密码& useUnicode = true & characterEncoding = UTF8
    • \n
\n
\n\n\n\n
 String url = "jdbc:mysql://localhost:3306/samp_db?" + "user=root&password=mysql&useUnicode=true&characterEncoding=UTF8";
\n\n\n\n

3、创建数据库连接

\n\n
\n
•要连接数据库，需要向java.sql.DriverManager请求并获得Connection对象，该对象就代表一个数据库的连接。 \n•使用DriverManager的getConnectin(String url , String username , String password )方法传入指定的欲连接的数据库的路径、数据库的用户名和密码来获得。\n
\n
\n\n\n\n
 String url = "jdbc:mysql://localhost:3306/test" ; \n String username = "root" ; \n String password = "root" ; \n try{ \n\n Connection con = DriverManager.getConnection(url , username , password ) ; \n\n }catch(SQLException se){ \n System.out.println("数据库连接失败！"); \n se.printStackTrace() ; \n }
\n\n\n\n

4、创建一个Statement

\n\n
\n
•要执行SQL语句，必须获得java.sql.Statement实例，Statement实例分为以下3 种类型： \n 1、执行静态SQL语句。通常通过Statement实例实现。 \n 2、执行动态SQL语句。通常通过PreparedStatement实例实现。 \n 3、执行数据库存储过程。通常通过CallableStatement实例实现。\n
\n
\n\n\n\n
 Statement stmt = con.createStatement() ; \n\n PreparedStatement pstmt = con.prepareStatement(sql) ; \n\n CallableStatement cstmt = con.prepareCall("{CALL demoSp(? , ?)}") ;

\n • ResultSet包含符合SQL语句中条件的所有行，并且它通过一套get方法提供了对这些行中数据的访问。
\n • 使用结果集（ResultSet）对象的访问方法获取数据：

\n

7、关闭JDBC对象

\n\n
\n
 操作完成以后要把所有使用的JDBC对象全都关闭，以释放JDBC资源，关闭顺序和声明顺序相反： \n 1、关闭记录集 \n 2、关闭声明 \n 3、关闭连接对象\n
\n
\n\n\n\n
 if(rs != null){  // 关闭记录集 \n try{ \n rs.close() ; \n }catch(SQLException e){ \n e.printStackTrace() ; \n } \n } \n\n if(stmt != null){  // 关闭声明 \n try{ \n stmt.close() ; \n }catch(SQLException e){ \n e.printStackTrace() ; \n } \n } \n\n if(conn != null){  // 关闭连接对象 \n try{ \n conn.close() ; \n }catch(SQLException e){ \n e.printStackTrace() ; \n } \n }
