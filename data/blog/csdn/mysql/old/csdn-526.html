约束：
\n1) not null (非空), 只能用于列级约束
\n2) unique(唯一)
\n3) primary key(主键--非空且唯一) 表中只允许一个主键
\n4) foreign key(外键--引用别的表的数据)
\n5) check, 只能用于列级约束
\n6) default 缺省值

\n

 语法描述：
\n         对于列级约束，直接写在列定义的后边就行。
\n         列定义 not null (非空约束)
\n         表级约束，在列定义完后，使用关键字
\n         约束关键字(列名)
\n 
\n 当主键约束为两个列时只有两个列连接值一样的时候才会违反约束。

\n       添加
\n       alter table 表名

设置级联删除
\n 语法描述：
\n     设置外键，在删除母表记录时，子表记录会被级联，也被删除
\n     alter table 表名
\n     add [constraint 键名] foreign key(子列引用的列) references 母表(母表主键) on delete cascade on update cascade;
\n  
\n     设置外键，在删除母表记录时，子表记录中的引用列被置为null
\n     alter table 表名
\n      add [constraint 键名] foreign key(子列引用的列) references 母表(母表主键) on delete set null on update set null;

\n
小例子：\n\t\t级联删除\n\t\talter table classes \n\t\tadd constraint my_fk foreign key(master) references teachers(id) on delete cascade on update cascade; \n\t\t级联置null\n\t\talter table classes \n\t\tadd constraint my_fk foreign key(master) references teachers(id) on delete set null on update set null; \n\t
