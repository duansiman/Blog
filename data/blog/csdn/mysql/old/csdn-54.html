\nLEFT()和RIGHT()
\n
\n分别有两个参数，第一个为所用的字符串，第二个为取几位
\n
\n例如获取MySQL的前两位
\n
mysql> SELECT LEFT('MySQL',2);\n+-----------------+\n| LEFT('MySQL',2) |\n+-----------------+\n| My |\n+-----------------+\n1 row in set (0.00 sec)

\n
\n例如获取MySQL的后两位
\n
mysql> SELECT RIGHT('MySQL',2);\n+------------------+\n| RIGHT('MySQL',2) |\n+------------------+\n| QL |\n+------------------+\n1 row in set (0.00 sec)

\n
\nLENGTH获取字符串长度
\n
mysql> SELECT LENGTH('MySQL');\n+-----------------+\n| LENGTH('MySQL') |\n+-----------------+\n| 5 |\n+-----------------+\n1 row in set (0.02 sec)

\nLTRIM,RTRIM,TRIM
\n
\n比如直接用SELECT TRIM(' mysql ');
\n
\n将会删除mysql的前和后的空格，LTRIM只删除左侧的空格，RTRIM只删除右侧的空格。
\n
\n
\n
\n
\n其他用法：利用TRIM()删除左右两侧的特定字符。
\n
\n例如：
\n
mysql> SELECT TRIM(LEADING '?' FROM '??MySQL???');\n+-------------------------------------+\n| TRIM(LEADING '?' FROM '??MySQL???') |\n+-------------------------------------+\n| MySQL??? |\n+-------------------------------------+\n1 row in set (0.02 sec)\nmysql> SELECT TRIM(TRAILING '?' FROM '??MySQL???');\n+--------------------------------------+\n| TRIM(TRAILING '?' FROM '??MySQL???') |\n+--------------------------------------+\n| ??MySQL |\n+--------------------------------------+\n1 row in set (0.00 sec)\nmysql> SELECT TRIM(BOTH '?' FROM '??MySQL???');\n+----------------------------------+\n| TRIM(BOTH '?' FROM '??MySQL???') |\n+----------------------------------+\n| MySQL |\n+----------------------------------+\n1 row in set (0.01 sec)

\n

\n

\n
\n注：TRIM()只能删除前导和后续的，不能删除字符中间的，比如TRIM('My SQL');这个空格就删不掉。

\nSUBSTRING()字符串截取
\n
\n所含参数，从第几位截，截取几位。
\n
mysql> SELECT SUBSTRING('MySQL',1,2);\n+------------------------+\n| SUBSTRING('MySQL',1,2) |\n+------------------------+\n| My |\n+------------------------+\n1 row in set (0.00 sec)

\n
\n需要注意的是，和编程语言不同的是，第一位是1，不是0。
\n
\n如果只有从第几位起，没有截取几个，就会，一直截到最后。
\n
mysql> SELECT SUBSTRING('MySQL',3);\n+----------------------+\n| SUBSTRING('MySQL',3) |\n+----------------------+\n| SQL |\n+----------------------+\n1 row in set (0.00 sec)

\n
\n位置也可以为负值（倒着数）
\n
mysql> SELECT SUBSTRING('MySQL',-1);\n+-----------------------+\n| SUBSTRING('MySQL',-1) |\n+-----------------------+\n| L |\n+-----------------------+\n1 row in set (0.04 sec)

\n
\n注，只能是起始位置可以有负值，但是截取长度不能有负值。
\n
