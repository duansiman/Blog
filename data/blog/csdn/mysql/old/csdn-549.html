2 问题分析

\n1）假如system_user只有10行数据(u1,u2,u3,u4,u5,u6,u7,u8,u9,u10)。
\n2) 这时候每次取5行。分2两次取。
\n3）正常情况第一页取(u1,u2,u3,u4,u5)，正常情况第二页取(u6,u7,u8,u9,u10)。
\n4）如果在第一页去完，第二页取之前的时间间隙里面。u2,u3被删除了。
\n5）那么第二页取的时候。数据变成了(u1,u4,u5,u6,u7,u8,u9,u10)。
\n6）那么取出来第二页的数据就变成了（u8,u9,u10）。
\n

7）u6,u7被丢失了。

\n

3 解决方案

\n每次取的时候where带着最大偏移量。如下面的代码。
\n
$systemUserTable = new systemUserTable();\n \n// 每次取20000\n$pageSize = 20000;\n\n\n// 每页最大编号\n$pageMaxId = 0;\n\n\nwhile ( 1 ) {\n $sql = "select userid from system_user where userid > ${pageMaxId} order by userid asc limit " . $pageSize;\n \n //取数据\n $rows = $fyHouse->fetchAll($sql);\n \n // 没有数据，结束\n if (empty($rows)) {\n break;\n }\n \n // 遍历数据\n foreach ( $rows as $key => $val ) {\n $userid = $val['userid'];\n \n // 记录当前最大分页编号\n if ($userid > $pageMaxId) {\n $pageMaxId = $userid;\n }\n \n echo $userid."\\r\\n";\n }\n}

\n过程分析：
\n1）system_user只有10行数据(u1,u2,u3,u4,u5,u6,u7,u8,u9,u10)。
\n2) 第一次取，最大偏移量0,limit 5;则取出(u1,u2,u3,u4,u5)。记录最大的偏移量u5的值。
\n4）如果在第一页去完，第二页取之前的时间间隙里面。u2,u3被删除了。
\n5）那么第二页取的时候。数据变成了(u1,u4,u5,u6,u7,u8,u9,u10)。
\n

3）第二次取，最大偏移量u5,limit 5;则取出(u6,u7,u8,u9,u10)。记录最大的偏移量u10的值。
