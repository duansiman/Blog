存储过程和函数

\n\n

存储过程和函数的定义：

\n\n
事先通过编译并存储在数据库中的一段sql语句的集合。\n
\n\n

存储过程和函数的区别：

\n\n
函数必须有返回值，存储过程必须有\n函数从参数只能是in，存储过程的参数可以是 in out inout \n
\n\n

创建存储过程或者函数

\n\n

创建简单的存储过程

\n\n
无参数的存储过程\n```\ndelimiter //\ndrop procedure if exists my_proce11//\n create procedure mysp11() select now()//\n```\n\n调用无参数的存储过程 \n call mysp11();\n
\n\n

创建含有输入输出的存储过程

\n\n\n\n
 delimiter //\n\n drop procedure if exists myprocedure//\n create procedure myprocedure(in pid int,in age int,out pcount int)\n sql security definer\n begin\n select name from country where id=pid and user_age=age;\n select FOUND_ROWS() INTO pcount;\n end\n //\n\n call myprocedure(2,3,@z)//\n select @z;

\n\n

定义

\n\n

\n\n
create trigger trigger_name trigger_time trigger_event on tabname for\neach row trigger_stmt\n\ntrigger_time 可以是after或者before\ntrigger_event可以是insert after before\n
