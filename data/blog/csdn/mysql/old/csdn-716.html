1、存储过程的概念

\n\n
\n

在一些语言中，如pascal，有一个概念叫“过程”procedure，和“函数”function，在php中，没有过程，只有函数。

\n
\n\n

过程：封装了若干条语句，调用时，这些封装体执行
\n函数：是一个有返回值的“过程”
\n总结：过程是一个没有返回值的函数

\n\n

在MySQL中：
\n我们把若干条sql封装起来，起个名字 —— 过程
\n把此过程存储在数据库中 —— 存储过程

\n\n

2、创建存储过程

\n\n
create procedure procedureName()\nbegin\n //--sql 语句\nend$
\n\n

3、查看已有的存储过程

\n\n\n\n
show procedure status
\n\n

4、删除存储过程

\n\n\n\n
drop procedure procedureName;
\n\n

5、调用存储过程

\n\n\n\n
call procedureName();
\n\n

注意：我这里已经将MySQL的结束标识符改为$，如果要知道怎么设置为$，请参考我的另一篇文章：MySQL触发器。

\n\n
create procedure p1()\nbegin\n select 2+3;\nend$

7、引入变量

\n\n

存储过程是可以编程的，意味着可以使用变量，表达式，控制结构来完成复杂的功能，在存储过程中，用declare声明变量：

\n\n
declare 变量名 变量类型 [default 默认值]
\n\n

使用：

\n\n
create procedure p2()\nbegin\n declare age int default 18;\n declare height int default 180;\n select concat('年龄：',age,'身高：',height);\nend$
\n\n

8、引入表达式

\n\n

\n\n
set 变量名:= expression
\n\n

使用：

\n\n
create procedure p3()\nbegin\n declare age int default 18;\n set age := age + 20;\n select concat('20年后年龄：',age);\nend$

12、存储过程参数的输入输出类型

\n\n
\n

主要有in、out、inout三种类型

\n \n

需求：从1加到N

\n
\n\n

输入型的数据是我们给出值，输出型是我们给出变量名，用于乘装输出的变量值。

\n\n

（1）in型，此时in为输入行参数，它能接受到我们的输入

\n\n
create procedure p7(in n int)\nbegin\n declare total int default 0;\n declare num int default 0;\n while num <= n do\n set total := total + num;\n set num := num + 1;\n end while;\n select total;\nend$
\n\n

调用：

\n\n
call p7(100);
\n\n

输出结果：

\n\n

\n\n

（2）out类型的参数

\n\n
create procedure p8(in n int,out total int)\nbegin\n declare num int default 0;\n set total := 0;\n while num <= n do\n set total := total + num;\n set num := num + 1;\n end while;\nend$
\n\n

调用：

\n\n
call p8(100,@total); --100为输入型参数，而@total为输出型变量\nselect @total; --输出@total变量
\n\n

输出结果：

\n\n

\n\n

（3）inout类型的参数

\n\n
create procedure p9(inout age int)\nbegin\n set age := age+20;\nend$
\n\n

调用：

\n\n
set @age = 18; --设置@age变量为18\ncall p9(@age); --调用p9存储过程，@age变量为实参\nselect @age; --显示@age变量
\n\n

输出结果：

\n\n

\n\n

inout型变量的实参也是一个变量名，这个变量在存储过程中既作为输入变量，又作为输出变量。
