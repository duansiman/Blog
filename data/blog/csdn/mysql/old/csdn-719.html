         在超哥的帮助下，完成了基于InnoDb数据引擎的mysql数据库集群搭建，实现了主从复制的功能，本篇博文介绍如何使用二进制安装mysql的方法，具体实现步骤如下：

\n


\n

\n

软件使用说明：
\nLiunx系统：centOS 7
\n数据库版本：mysql-5.6.12-linux-glibc2.5-x86_64.tar
\n
\n
\n1、 下载centos7新装系统需要的文件：
\n
\n
\nyum install libaio
\nyum install 'perl(Data::Dumper)'
\nyum -y install perl perl-devel
\n
\n
\n2、 进入用户家目录创建tools文件夹
\n
\n
\n# cd ~
\n#mkdir tools
\n#cd tools
\n
\n
\n3、 下载MySQL5.6二进制包
\n下载wget工具：
\nrpm ivh wget-1.12-1.4.el6.x86_64.rpm
\nwget http://cdn.mysql.com//archives/mysql-5.6/mysql-5.6.12-linux-glibc2.5-x86_64.tar.gz
\n（也可以从云盘下载：http://pan.baidu.com/s/1i44FJWL）
\n
\n
\n3、添加mysql用户和用户组
\n# groupadd mysql
\n# useradd -g mysql mysql
\n
\n
\n4、解压二进制安装包
\n# tar zxvf  mysql-5.6.12-linux-glibc2.5-x86_64.tar.gz
\n
\n
\n5、创建/application/data和/application/mysql文件夹
\n# mkdir /application/data -p
\n
\n
\n6、将解压好的mysql-5.6.12-linux-glibc2.5-x86_64重命名为mysql 
\n# mv mysql-5.6.12-linux-glibc2.5-x86_64 /application
\n# mv /application/mysql-5.6.12-linux-glibc2.5-x86_64/ /application/mysql
\n
\n
\n7、修改mysql文件夹的用户和用户组为mysql
\n# cd /application/mysql
\n#chown -R mysql:mysql ./
\n
\n
\n8、mysql已经安装完毕，下面要初始化mysql表、information表等
\n#scripts/mysql_install_db --user=mysql --basedir=/application/mysql --datadir=/application/data
\n
\n
\n9、设置启动mysqld和mysql的配置文件
\n# cp /application/mysql/support-files/mysql.server /etc/init.d/mysqld
\n# cp /application/mysql/my.cnf /etc/my.cnf
\n
\n
\n10、修改配置文件
\n# vi /etc/my.cnf
\n
\n
\nbasedir =/application/mysql 
\ndatadir = /application/data
\nport = 3306
\nserver_id = 1
\n
\n
\n11、创建软链接启动mysql
\n# ln -s /application/mysql/bin/mysql /usr/bin
\n
\n
\n12、设置mysql开机自启
\n# chkconfig mysqld on
\n
\n
\n13、关闭防火墙
\n# systemctl start firewalld.service
\n# systemctl stop firewalld.service
\n# systemctl disable firewalld.service  禁止开机启动
\n
\n
\n14.启动服务：
\n# service mysqld start
\n
\n
\n15.创建用户：
\n

\n

#mysql –uroot –p

\n
