       近来进行mysql的研究使用，对于服务器知识一窍不通的我还是需要从基本练起，首先从mysql的安装开始，安装步骤如下：

\n

        

\n

  一、 卸载原先数据库
\n
\n

\n

1、  查询系统有没有安装数据库
\nrpm -qa | grep mysql
\n
\n

\n

        2、卸载原先的数据库
\n                    rpm -e mysql　　// 普通删除模式
\n                    rpm -e --nodeps mysql　　// 强力删除模式，如果使用上面命令删除时，提示有依赖的其它文件，则用该命令可以对其进行强力删除

\n

       如果上述命令任然无法删除，则可以参考： 《linux 怎么完全卸载mysql数据库 》
\n
\n

\n

二、 使用yum 安装数据库

\n


\n       1、查询yum提供的数据库版本
\n                     yum list | grep mysql
\n 

\n

       2、安装数据库

\n

                    yum install  -y  mysql-server  mysql  mysql-deve
\n
\n

\n

       3、  查看安装的数据库版本
\n                     rpm -qi mysql-server
\n
\n

\n

       4、启动数据库
\n                    service mysqld start
\n            如果命令启动不了：mysql: unrecognized service
\n            使用：
\n                   /etc/rc.d/init.d/mysqld start
\n            修改启动文件权限
\n                   chmod 755 /etc/rc.d/init.d/mysqld
\n           

\n

            再次启动：service mysqld start
\n
\n        5、查询数据库是否开机启动
\n                      chkconfig --list | grep mysqld
\n
\n

\n

       6、设置开机启动
\n                      chkconfig mysqld on
\n 
\n       7、设置mysql 数据库 root 用户的初始密码
\n                     /usr/bin/mysqladmin -u root password 'new-password'
\n          或
\n                     mysqladmin -u root password 'root'

\n

      8、进入mysql：#mysql -u root

\n

\n

\n

 三、查看数据库配置文件位置

\n


\n

\n

       1、cat /etc/my.cnf

\n

           [root@bogon Desktop]# cat /etc/my.cnf
\n                [mysqld]
\n                datadir=/var/lib/mysql
\n                socket=/var/lib/mysql/mysql.sock
\n                user=mysql
\n                # Disabling symbolic-links is recommendedto prevent assorted security risks
\n                symbolic-links=0

\n


\n

\n

2、查看mysql数据位置与日志

\n

              /var/lib/mysql

\n

             /var/log/mysqld.log

\n


\n

四、报错修改

\n

      

\n

          使用root用户登录发生以下错误：

\n

 mysqlaccess denied for user 'root'@

\n

      

\n

         进行如下修改：

\n

        # /etc/init.d/mysql stop
\n# mysqld_safe --user=mysql --skip-grant-tables --skip-networking &
\n# mysql -u root mysql
\nmysql> UPDATE user SET Password=PASSWORD('newpassword') where USER='root';
\nmysql> FLUSH PRIVILEGES;
\nmysql> quit
\n# /etc/init.d/mysql restart
\n# mysql -uroot -p
\nEnter password: <输入新设的密码newpassword>
