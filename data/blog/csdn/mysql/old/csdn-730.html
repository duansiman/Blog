    文章出处：基于CentOS的Mysql学习补充二--使用Shell创建数据库

\n

    关于使用基于命令使用mysql数据库的方法可见文章《基于CentOS的Mysql的使用说明》，本文只要介绍如何使用Shell脚本来创建我们想要的数据库，下面给出我创建数据库的Shell脚本，基本也就是将mysql的一些命令放在shell脚本里面而已。

\n

\n
#!/bin/bash\n\n#Author: chisj\n#Time: 2015.7.22\n#Describe: Create Database\n\n#The username of mysql database\nUSER="root"\n\n#The password of mysql database\nPASS="dragonwake"\n\n#The datebase name will be created\nDATABASE="SmartCare"\n\nmysql -u $USER -p$PASS << EOF >/dev/null\n\nCREATE DATABASE $DATABASE\n\nEOF

\n这里说明几点需要注意的问题 \n

\n

1、如果是在windows下面编辑上面这个脚本，然后在linux下面去执行，可能会出现这个问题

\n


\n

\n

从上图也可以看到我这里的解决方法，使用命令dos2unix将windows文件格式转化为linux（unix）文件格式

\n

#dos2unix filename

\n

当然前提是需要你的系统安装了该命令，默认系统时不安装的，可以使用yum进行安装

\n

#yum install dos2unix
